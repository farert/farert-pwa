# メイン画面仕様

## 対象
- ルート: `/`
- 実装: `src/routes/+page.svelte`

## 画面の役割
- 現在編集中の経路を直接編集する主画面。
- 発駅設定、区間追加、区間削除、反転、保存、詳細確認、きっぷホルダ操作の起点になる。
- `mainRoute` の正本編集画面であり、他画面はこの画面の状態を補助する。

## レイアウト構成

### ヘッダー
- ハンバーガーボタン: `DrawerNavigation` を開く。
- タイトル: Farert とアプリ名を表示する。
- テーマ切替: ライト / ダークを即時反映し、`localStorage.theme` に保存する。
- 右上メニュー:
  - バージョン情報
  - ヘルプ
  - 条件付きの経路オプション

### メインコンテンツ
- 発駅カード
- 区間カード列
- 運賃サマリーカード
- 「+ 経路を追加」カード
- 画面下部固定の操作ナビ

### 補助 UI
- `DrawerNavigation`
- エラーバナー / 情報バナー
- 経路上書き確認ダイアログ

## 状態別の見え方
- 経路が空:
  - 発駅未設定表示を出す。
  - 区間カードは出さない。
  - 運賃サマリーは出さない。
- 発駅のみ設定済み:
  - 発駅カードを表示する。
  - 区間プレースホルダまたは追加導線を出す。
- 区間あり:
  - 各区間をカード列で表示する。
  - 運賃計算可能ならサマリーを表示する。
- 経路が終端:
  - 追加カードを無効化する。
  - 文言を「経路が終端に達しました」に切り替える。

## 主要操作
- 発駅カード押下: `/terminal-selection` へ遷移する。
- 区間カード押下: その区間までを URL 圧縮して `/detail` へ遷移する。
- 運賃サマリー押下: 全経路を `/detail` へ遷移する。
- 経路追加押下:
  - 末尾駅を基準に `/line-selection?from=main&station=...` へ遷移する。
  - 最終区間の路線が必要なら `line` クエリにも載せる。
- 戻る押下:
  - 区間があれば `removeTail()`
  - 区間がなければ `removeAll()`
- 反転押下: `reverse()` を実行する。
- 保存押下: `/save` へ遷移する。
- 上下ボタン: ページ先頭 / 末尾へスクロールする。

## きっぷホルダ連携
- 現在経路の追加は 2 区間以上のときのみ許可する。
- ドロワー共有は `routeScript` を改行区切りで共有する。
- ホルダ内経路の選択時、編集中経路が 2 区間以上あり内容も異なる場合は確認ダイアログを出す。
- 各カードでは運賃種別の切替、削除、並べ替えを行う。

## 経路オプション

### 表示位置
- 右上メニュー内の「経路オプション」セクション。

### 設計意図
- 経路追加前後のルート解釈差分を、別画面に行かずメイン画面上で切り替えられるようにする。
- ルートそのものの意味を変えるオプションなので、単なる表示設定ではなく `mainRoute` 側へ反映する。

### 大阪環状線オプション
- 表示条件:
  - 経路セグメントまたは `routeScript` に大阪環状線の痕跡がある
  - もしくは `isOsakakanDetourEnable()` が有効
- 表示文言:
  - 近回り時: `大阪環状線 近回り`
  - 遠回り時: `大阪環状線 遠回り`
- 操作:
  - `setDetour(next)` を呼ぶ
  - 成功後 `mainRoute.set(route)` で再反映する
- 失敗時:
  - エラーバナーを表示する
- 永続化と共有:
  - この設定は `routeScript` に反映される経路オプションとして扱う
  - そのため保存、詳細画面遷移、共有 URL でも保持される
  - 共有 URL では個別のクエリ項目を増やさず、圧縮された経路データの一部として `r` に含まれる

### 小倉-博多間別線扱い
- WASM API と状態取得処理は `isNotSameKokuraHakataShinZai()` / `setNotSameKokuraHakataShinZai()` を持つ。
- 設計上は、大阪環状線オプションと同様にメイン画面の経路オプションに属する。
- 仕様書としては以下のトグル情報を保持する。

| 状態 | 表示文言 | Setter |
|---|---|---|
| 同一路線扱い | `小倉-博多間新幹線在来線別線扱い（有効）` | `setNotSameKokuraHakataShinZai(true)` |
| 別線扱い | `小倉-博多間新幹線在来線別線扱い（無効）` | `setNotSameKokuraHakataShinZai(false)` |

## 設計上の注意
- 経路オプションは route の一時フラグではなく、`routeScript` と復元経路に影響しうる設定として扱う。
- そのため、メイン画面での切替後は詳細画面・保存画面・共有 URL に反映される前提で設計する。
