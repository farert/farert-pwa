# メイン画面の詳細仕様

## ヘッダー部

- ハンバーガーメニュー | タイトル | バージョン情報アイコン
- バージョン情報アイコンタップでバージョン情報表示画面に遷移

## リスト表示部（0〜N行）

### 行の構成

1. 1行目: 発駅表示
2. 2〜N-2行目: 路線と着駅（2行1組）
3. N-1行目: 「追加」ラベル
4. N行目（最終行）: 営業キロ、普通運賃、有効日数

### 初期状態

- 1行目: 「発駅を指定してください」のみ表示

### 発駅指定時の動作

1. 発駅選択画面を表示
2. 選択後、1行目が発駅表示に変更
3. 2行目に「追加」が表示される

### 経路追加時の動作

1. 路線選択画面 → 分岐駅選択画面の順で表示
2. 選択後、2行目以降に路線・着駅が追加される
3. N-1行目が「追加」、N行目が営業キロ等の表示となる
4. 最終駅まで追加可能（経路オブジェクトが最終駅判定を返す）
5. 画面がいっぱいになっても「追加」が隠れないようスクロール

## 行のタップ動作

### 1行目（発駅）

- 発駅選択画面を表示
- 選択時、既存の経路行を全削除し、1行目（発駅）+ 2行目（追加）の2行表示に戻る

### 2〜N-2行目（路線・着駅）

- タップした行を着駅とした結果詳細画面に遷移

### N-1行目（追加）

- 路線選択画面 → 駅選択画面で経路追加

### N行目（最終行・営業キロ等）

- N-2行目と同じ動作（全経路の結果詳細表示画面）

## フッター部（画面下部・1行）

左から順にアイコン配置:

1. **戻る**: 初期状態はグレーアウト。経路1つ以上かつ発駅がある場合に有効
2. **反転**: 初期状態はグレーアウト。経路2つ以上かつ発駅・路線着駅がある場合に有効
3. **オプション**: グレーアウトで無効
4. **ファイルフォルダ**: 常に有効
