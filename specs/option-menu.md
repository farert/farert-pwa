# オプション仕様

## メイン画面のオプション
- 表示位置: メイン画面右上メニュー
- 表示条件: 大阪環状線オプションまたは小倉博多オプションが利用可能

### 項目
- `大阪環状線近回り` / `大阪環状線遠回り`
- `小倉-博多間新幹線在来線別線扱い（有効）`
- `小倉-博多間新幹線在来線別線扱い（無効）`

## 詳細画面のオプション
- 表示位置: 右上メニュー
- `FareInfo` の有効フラグに応じて項目を動的生成する

### 項目例
- 特例を適用する / しない
- 発駅を単駅指定 / 着駅を単駅指定
- 指定した経路で運賃計算 / 最安経路で運賃計算
- 規程 115 条切替
- JR 東海株主優待券適用切替

## 共通
- 切替は WASM の `Farert` インスタンスに対して行う。
- 切替後は再計算して画面表示を更新する。
- 設定値は個別の別ストアには持たず、現在経路の状態として扱う。
