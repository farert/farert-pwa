# バージョン情報表示画面の詳細仕様

## 画面レイアウト（縦方向中央配置）

### 1. アプリバージョン表示（最上部）

- **位置**: 上部マージン 32dp
- **配置**: センタリング
- **内容**: `Farert x.x.x`（アプリのバージョン番号）
- **データソース**: Bundle.main CFBundleShortVersionString

### 2. メインタイトル

- **位置**: アプリバージョンの下
- **配置**: センタリング
- **フォントサイズ**: 20dp
- **内容**: "バージョン情報" または "Version Info"

### 3. データベースバージョン情報

- **位置**: メインタイトルの下 32dp
- **配置**: センタリング
- **内容**: `DB Rev. [名称](作成日付)`
- **例**: `DB Rev. [2023年10月改正](2023-10-01)`
- **データソース**: WASMモジュールの `databaseVersion()` から取得
  - `dbverInf.name`: データベース名称
  - `dbverInf.create_date`: 作成日付

### 4. 消費税率情報

- **位置**: データベースバージョン情報の下 8dp
- **配置**: センタリング
- **内容**: `消費税: xx%`
- **データソース**: `dbverInf.tax`

### 5. 著作権表示

- **位置**: 消費税率情報の下 8dp
- **配置**: センタリング
- **内容**: 著作権情報テキスト

### 6. 使用許諾契約（EULA）

- **位置**: 著作権表示の下 24dp
- **配置**: センタリング
- **幅**: 280dp
- **内容**: 使用許諾契約の本文

### 7. サポートサイトURLボタン

- **位置**: EULAの下
- **配置**: センタリング
- **スタイル**: ボーダーレス・カラーボタン
- **テキスト**: サポートサイトURL（太字）
- **アクション**: タップでブラウザ起動
  - URL: `http://farert.blogspot.jp/`

### 8. 閉じるボタン

- **位置**: 画面最下部 8dp マージン
- **配置**: 横幅いっぱい
- **スタイル**: ボーダーレス・カラーボタン
- **フォントサイズ**: 大
- **テキスト**: "OK" または "閉じる"（太字）
- **アクション**: 画面を閉じてメイン画面へ戻る

## データ取得タイミング

- `viewWillAppear` / `onResume` 時にWASMモジュールから最新情報を取得
- データベースバージョン情報は動的に更新される

## 表示例

```
Farert 1.2.3

バージョン情報

DB Rev. [2023年10月改正](2023-10-01)
消費税: 10%

Copyright © 2015-2023 TAKEDA, Noriyuki

[使用許諾契約テキスト...]

[http://farert.blogspot.jp/]

[          OK          ]
```

## 技術実装メモ

- 静的な画面構成
- WASMモジュールの `cRouteUtil.databaseVersion()` から情報取得
- アプリバージョンはビルド時のメタデータから取得
- サポートURLはシステムブラウザで開く
